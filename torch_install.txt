#### install torch geometric
1. ## uninstall CPU版
   pip uninstall -y torch torchvision torchaudio torch-geometric torch-geometric-temporal torch-sparse torch-scatter torch-cluster torch-spline-conv pyg-lib  
2. ## install 最新GPU版
   pip install torch==2.8.0 torchvision==0.23.0 torchaudio==2.8.0 --index-url https://download.pytorch.org/whl/cu126
