#### install torch geometric
1. `pip uninstall -y torch torchvision torchaudio torch-geometric torch-geometric-temporal torch-sparse torch-scatter torch-cluster torch-spline-conv pyg-lib`  ## uninstall CPU版
2. `pip install torch==2.8.0 torchvision==0.23.0 torchaudio==2.8.0 --index-url https://download.pytorch.org/whl/cu126`  ## install 最新GPU版



####   install torch geometric
2. `py -c "import torch; print(torch.__version__)"` ## check依加版本
   \>\>\> 2.8.0+cu126
3. `py -c "import torch; print(torch.version.cuda)"` # check依加版本
   \>\>\> 12.6

4. `pip install pyg_lib torch_scatter torch_sparse torch_cluster torch_spline_conv -f https://data.pyg.org/whl/torch-2.8.0+cu126.html` ##  因為沒有2.9.0，所以install 2.8.0
